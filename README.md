# currency_converter_Hexlet
Hexlet homework

### Local setup
```shell
$ git clone git@github.com:Eira/currency_converter_Hexlet.git
$ cd currency_converter_Hexlet
$ python3.11 -m venv venv
$ source venv/bin/activate
$ pip install requests mypy flake8
```

### Local run
```
python -m main
mypy .
flake8 *.py
```
